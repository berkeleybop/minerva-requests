new request_set(user_token[, model_id])

Constructor for a Minerva request item set.

Handle sets of requests and serialize for Minerva call.

Request sets are essentially serial request queues, that reference eachother using the request_variables contained in invididual requests.

As the request_set operations almost always produce request_sets (with senisible defaults and fail modes), they can easily be chained together.

If a model_id is given, it will be applied to any request that does not have one.
